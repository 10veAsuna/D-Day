# 二轮游戏

## 一、基础信息

### 名称

- 中文名：钢铁浪潮·诺曼底
- 英文名：Steel Tide: Normandy

### 类型

- 核心类型：**第一人称射击（FPS）+ 大战场战术对抗**
- 细分标签：二战登陆战、抢滩突击、沉浸式战场

### 核心定位

“1944年6月6日凌晨，你作为美军第1步兵师（大红一师）的列兵，挤在摇晃的登陆艇里，随着潮水逼近奥马哈海滩——前方是德军密布的机枪堡、铁丝网和地雷，身后是滔滔海水。你的任务：活着冲上滩头，夺取第一块立足之地。”

### 核心玩法

玩家通过 **“抢滩→突破→巩固”三阶段** 体验诺曼底登陆的残酷：

- **抢滩阶段**（0-5分钟）：从登陆艇跳入齐胸海水，躲避德军岸防炮和机枪扫射（子弹打起水花），艰难爬向倾斜的沙滩；
- **突破阶段**（5-15分钟）：利用沙袋/残骸掩护，迂回接近德军碉堡（或呼叫舰炮支援摧毁），夺取第一道防线；
- **巩固阶段**（15分钟后）：与后续增援部队汇合，防守滩头据点，抵御德军装甲车反扑。

### 世界观设定

- **时间背景**：1944年6月6日（D-Day），诺曼底奥马哈海滩（美军主攻区域）；
- **地点背景**：从登陆艇停泊点到滩头斜坡（覆盖“血腥100米”核心区），包含防波堤、铁丝网、碉堡群、登陆艇残骸；
- **背景故事**：你是二等兵杰克，所在的E连负责攻占“Easy Red”滩头段。上级命令：“不惜一切代价在日出前拿下碉堡——那是后续部队的生命线。”

## 玩法与系统设计

### 基础玩法

- **核心行动**：射击（步枪/冲锋枪）、卧倒（降低受击伤害）、涉水移动（速度降低且子弹水花暴露位置）、掩体交互（躲沙袋后换弹）；
- **目标类型**：主线目标（必做，如“夺取滩头碉堡”）、可选目标（零击杀/呼叫舰炮支援/救援被压制队友）。

### 游戏模式

- **主线模式**：单章节“奥马哈海滩抢滩战”（完整还原D-Day当日06:30-08:00的战斗）；
- **隐藏内容**：收集“战友日记”解锁背景故事（如E连士兵的家书、德军守军日志）。

### 玩家目标

- **短期目标**：活着冲上滩头，找到掩体躲避机枪火力；
- **长期目标**：通过策略（迂回/火力压制/呼叫支援）夺取碉堡，影响后续部队登陆成功率。

### 核心机制

| **机制**       | **规则**                                                     |
| :------------- | :----------------------------------------------------------- |
| **视野与隐蔽** | 德军机枪手视野覆盖沙滩斜坡（玩家直立移动必被发现），卧倒或利用沙袋遮挡可降低暴露； |
| **载具火力**   | 德军岸防炮（150mm）和机枪堡（MG42）形成交叉火力网，玩家需找到死角（如防波堤后方）； |
| **舰炮支援**   | 按Q键标记敌方碉堡，舰炮会在30秒后轰炸（弹道延迟，需提前预判）； |
| **小队AI**     | 5名队友（医疗兵/机枪手/步枪手）会跟随玩家，医疗兵可复活倒地队友（限1次/章）； |
| **环境互动**   | 推倒木箱制造路障阻挡机枪火力，打爆油桶引发爆炸清场，利用登陆艇残骸作掩体。 |

### 基础操作

| **动作**  | **按键（示例）**  | **说明**                                 |
| :-------- | :---------------- | :--------------------------------------- |
| 移动      | WASD              | 涉水时速度降低50%，跑步易暴露位置        |
| 卧倒/站立 | Ctrl/Space        | 卧倒时射击精度提高，但移动极慢           |
| 射击      | 鼠标左键          | 射击后需手动换弹（不同武器弹匣容量不同） |
| 交互      | E                 | 开门/拾取弹药/呼叫舰炮支援               |
| 载具操作  | F（接近缴获机枪） | 操作MG42压制敌军（固定位置射击）         |
| 跑步      | shift             | 按住shift速度更快                        |

## **系统模块设计**

### **1. 任务系统**

- **任务简报**：战前通过无线电接收目标（如“07:00前夺取Easy Red段碉堡”）；
- **进度追踪**：UI显示当前目标（如“接近防波堤”“摧毁机枪堡”）、队友存活状态；
- **结局分支**：根据玩家行为（是否呼叫支援/击杀数/队友存活）生成不同结局（如“惨胜”“全员牺牲”）。

### **2. 战场系统**

- **多阶段流程**：海滩（涉水）→斜坡（掩体战）→碉堡（室内争夺）→内陆（巩固据点）；
- **动态AI密度**：海滩初期AI密集（模拟“万箭齐发”），突破后逐渐稀疏。

### **3. 潜行与战术（可选）**

- **迂回路线**：通过悬崖下方或防波堤缝隙绕过主火力区（但可能遭遇德军巡逻队）；
- **隐蔽行动**：趴在沙滩上缓慢移动（几乎无声），接近敌人后背刺（仅限无警戒状态）。

### **4. 关键系统规则**

- **舰炮支援**：标记后需保持标记点30秒（期间被敌人发现会取消），轰炸误差±10米；
- **载具交互**：缴获的MG42有弹药限制（200发），过热后需冷却10秒；
- **小队指令**：按Q键下达“掩护我”“跟我冲”指令（队友AI会根据指令调整行为）。

## **美术与音效设计**

### **1. 美术风格**

- **整体基调**：二战写实风，偏灰绿色调（泥泞沙滩/锈蚀装备），突出“血腥与压抑”；
- **场景细节**：
  - 海滩：倾斜的沙地（行走困难）、铁丝网（缠绕尸体）、登陆艇残骸（燃烧冒烟）；
  - 碉堡：混凝土结构（弹痕累累），机枪口喷出火舌，内部有德军士兵尸体；
  - 海面：登陆艇颠簸（溅起水花），远处舰炮轰鸣掀起巨浪。
- **角色设计**：美军士兵穿着沾满泥浆的作战服，德军穿着灰色制服（部分戴钢盔），面部表情紧张/痛苦。

### **2. 音效与音乐**

- **环境音**：海浪声（持续背景）、登陆艇引擎轰鸣、士兵喊叫声（“Go！Go！Go！”）、德军无线电指令（“Feuer!”开火）；
- **交互音**：
  - 枪声（M1加兰德清脆，MG42高频密集）、子弹打水花的“噗噗”声、手雷爆炸的“轰隆”声；
  - 脚步声（沙滩上沉重模糊，涉水时带水声）、铁丝网刮擦声。
- **背景音乐**：
  - 抢滩阶段：低沉紧张的弦乐（营造绝望感）；
  - 突破阶段：急促鼓点（提升压迫感）；
  - 结局阶段：悲壮交响乐（呼应“惨胜”主题）。

## **任务结构**

### **任务1：抢滩登陆（核心目标：活着冲上滩头）**

**背景**：1944年6月6日，你作为美军第1步兵师士兵，挤在登陆艇内，随第一批部队冲向奥马哈海滩。前方是德军密集的机枪扫射和岸防炮，你的首要任务是 **躲过致命火力，抵达滩头斜坡的掩体**（如沙袋/残骸），为后续突破争取机会。

**核心玩法**：

- **生存优先**：躲避德军岸防炮爆炸（大范围伤害）和MG42机枪扫射（红色警示区，子弹打水花暴露位置）；
- **移动策略**：涉水移动速度降低50%，可选择 **“直冲斜坡”**（主流但危险）或 **“隐蔽爬行”**（绕防波堤缝隙，更安全但耗时）；
- **基础交互**：利用海浪/沙袋短暂掩护，接近滩头后找到掩体（如倒塌的木箱/混凝土块）躲避火力。

**任务目标**：

- 必做：抵达滩头斜坡掩体（血量≥30%，否则可能无法突破）；
- 可选：零击杀（仅躲避）、救助倒地队友（医疗兵AI存活）。

------

### **任务2：突破防线占领据点（核心目标：夺取德军机枪堡/碉堡）**

**背景**：成功上岸后，你发现滩头被德军机枪堡（MG42）和铁丝网封锁，必须 **突破这道防线，占领关键据点（碉堡）**，为后续部队打开通道。

**核心玩法**：

- **火力压制**：德军机枪堡射程25米，玩家直立移动必被击中，需利用掩体（沙袋/残骸）迂回接近；
- **突破策略**：
  - **正面突击**（高风险）：冲向碉堡下方，投掷手雷炸毁机枪堡（需贴近爆炸范围）；
  - **迂回战术**（低风险）：绕至碉堡侧面死角（无子弹覆盖），用缴获的MG42或冲锋枪压制守军；
  - **支援呼叫**（策略辅助）：按Q键标记碉堡，呼叫舰炮支援（30秒后轰炸，可能误伤玩家）。
- **环境交互**：可推倒木箱制造路障阻挡机枪火力，打爆油桶引发爆炸清场。

**任务目标**：

- 必做：摧毁/占领德军机枪堡（击杀守军或破坏关键设备）；
- 可选：零击杀（仅用投掷物/环境交互）、全员存活（AI队友不死亡）。
